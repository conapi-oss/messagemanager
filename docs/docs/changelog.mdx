# üìù Changelog

Below are the changelogs for the latest releases of the Message Manager:

4.1.2 - 2024-12-13
- Added support for macOS ARM64 and x86_64 (dmg)
- Minor UI improvements
- Python related fixes on the scripting support

4.1.1 - 2024-11-13
- Minor fixes to scenarios

4.1.0 - 2024-09-13
- Generic Scripting Support (Python and JEXL)
- AWS SQS improvements
- Minor UI improvements
- Updated documentation
- Automatic update checks are now only performed once every x days

4.0.6 - 2024-08-28
- Solace library update
- Solace drag and drop support (forward message to another queue)

4.0.5 - 2024-08-09
- Support Inverse Filtering
- Fixed issue with message deletion not working correctly for topic subscribers
- Keep selection in message details tree when deleting messages
- Fixed automatic module class loading

4.0.4 - 2024-08-08
- Fixed ActiveMQ Credentials Handling
- Allow alternate URLs to be configured if connection to the broker fails (i.e. SSH tunnels)
- Kafka: Partition and offset support for Consumer
- Improved search usability
- Allow filtering of messages using the search criteria
- Metadata support for messages

4.0.3 - 2024-07-29
- Allow SASL mechanism selection for Kafka connections
- Show asynchronous errors using an alert box
- Updated documentation

4.0.2 - 2024-07-26
- Allow Security Protocol selection for Kafka connections
- Updated documentation structure

4.0.1 - 2024-07-24
 - Show improved error messages for class loading issues and link to the documentation
 - First upload of the documentation

4.0 - 2024-07-20
 - Initial release supported by conapi GmbH
 - Support for Java 17
 - Java 9+ module support
 - Dark mode support
 - Automatic updates
 - AWS SQS support (commercial only)
 - Apache Kafka support (commercial only)
