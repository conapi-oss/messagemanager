# Import/Export of Messages

Message Manager allows you to import (send from file) and export messages from and to a queue or topic.
This is useful for testing and debugging purposes, moving messages between queues or topics, or for troubleshooting.

## Exporting Messages

To export messages, select the queue or topic.
Then select the messages you want to export and click the "Save" button.

## Importing Messages

You can import messages from a file or from folder and send them. 
See Message Sender for more details.

:::tip

You can also use the Scripting feature to import/export messages.

:::
