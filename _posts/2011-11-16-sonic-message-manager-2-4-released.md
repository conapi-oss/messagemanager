Sonic Message Manager 2.4 has just been released and adds support for Sonic MQ 8.5. This version contains the following changes and bugfixes:

  * 159: timestamps on saving messages?
  * 163: Startup fails: error saving prefs
  * 165: SMM does not connect when using #ONLY
  * 166: Use separate credentials when connecting to message broker
  * 167: Ability to specify Login SPI for LDAP logins
  * 168: Send settings are ignored when sending esbmsg
  * 169: Support for SonicMQ 8.5
