  * [148: Refresh queue list does not add new queues](http://queuemanager.nl/trac/ticket/148)
  * [149: Limited admin privs are unsupported](http://queuemanager.nl/trac/ticket/149)
  * [150: Error when Using a topic in JMSReplyTo field](http://queuemanager.nl/trac/ticket/150)
  * [151: Setting Time To Live doesn&#8217;t work with message sender](http://queuemanager.nl/trac/ticket/151)
