Message Manager

Download

Message Manager is a Java Web Start application, click below to start:

Message Manager 3.2

Java Web Start will provide you with updated versions automatically. Message Manager requires Java 7 or higher and runs on Windows, Linux or Mac OS X.

Before you download

Please consider donating a small amount to assure continued development of Message Manager. To donate, click the "Donate" button on the right. If you use Message Manager in your company and consider it valuable, please, let your manager(s) know and encourage them to donate on behalf of your company.

Nightly snapshots

A nightly build is also available at Message Manager 3.x-SNAPSHOT

Note: These builds are unstable and may or may not work!

Source code and issue tracker

The source code for this program is available on BitBucket. Please feel free to fork, clone and/or create issues.

https://bitbucket.org/gerco/messagemanager

Older versions

Sonic Message Manager 2.6 is still available for Sonic™ 6.1, 7.0, 7.5, 7.6, 8.0, 8.5 and 2013. Click your version number to download. This version is no longer supported and will not receive any more updates.
